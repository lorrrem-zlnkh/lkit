- Ветка `gh-pages` запушена в GitHub.

Ожидаемый Pages URL:

`https://lorrrem-zlnkh.github.io/lkit/`

Статус:

- URL на момент проверки отдавал `404`.
- Причина: GitHub Pages ещё не включён в настройках репозитория или не выбран источник публикации.
- Так как `gh` CLI не установлен, GitHub connector не видел репозиторий, а доступного HTTPS-токена в git credentials не было, включить Pages через API не получилось.
- Была открыта страница настроек:

`https://github.com/lorrrem-zlnkh/lkit/settings/pages`

Нужно выбрать:

- Source: `Deploy from a branch`
- Branch: `gh-pages`
- Folder: `/ (root)`
- Save

После этого GitHub Pages должен опубликовать сайт за несколько минут.
